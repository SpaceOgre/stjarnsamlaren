# Stjärnsamlaren

Ett litet webbläsarspel byggt tillsammans.

Spela genom att öppna `index.html` i webbläsaren. Hunden samlar stjärnor, undviker eldklotet och vinner vid 10 poäng.

## Filer

- `index.html` - spelets sida
- `style.css` - färger och layout
- `script.js` - spelregler och ritning
- `favicon.svg` - stjärnan på webbläsarfliken

## GitHub Pages

Det här projektet kan publiceras direkt med GitHub Pages från `main` och `/root`.
